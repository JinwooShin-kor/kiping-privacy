개인정보처리방침

Kiping (키핑) - AI 뉴스 키워드 모니터링
시행일: 2026년 2월 22일 | 최종 수정일: 2026년 2월 22일

Kiping(이하 "서비스")은 이용자의 개인정보를 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 개인정보처리방침은 서비스가 수집하는 개인정보의 항목, 수집 목적, 보관 기간 및 이용자의 권리에 대해 안내합니다.

1. 수집하는 개인정보 항목

구분 수집 항목 수집 방법
회원가입 이메일 주소, 이름(닉네임), 프로필 사진 URL Google/Kakao 소셜 로그인
서비스 이용 등록 키워드, 키워드 상세 설명, 관심 분야, 뉴스 열람 기록, 좋아요/댓글 활동 앱 내 직접 입력 및 자동 수집
알림 FCM 푸시 토큰, 알림 설정(ON/OFF, 빈도, 조용한 시간대) 기기에서 자동 생성
커뮤니티 게시글 제목/내용, 업로드 이미지/영상, 해시태그 앱 내 직접 입력
자동 수집 기기 정보(OS 종류), 앱 사용 통계(화면 방문 기록), 활동 히트맵 서비스 이용 시 자동 수집

Kiping은 주민등록번호, 결제 정보, 위치 정보 등 민감한 개인정보를 수집하지 않습니다.

2. 개인정보의 수집 및 이용 목적

목적 상세 설명
회원 관리 회원 식별, 로그인 인증, 계정 관리
맞춤형 뉴스 제공 등록된 키워드 기반 뉴스 매칭 및 피드 생성
AI 분석 뉴스 기사의 키워드 관련도 분석, 일일 AI 리포트 생성, 홈 인사이트 생성
알림 발송 키워드 매칭 뉴스 실시간 푸시 알림, 인앱 알림
커뮤니티 운영 게시글 작성/관리, 좋아요/댓글 기능, 그룹 운영
서비스 개선 사용 통계 분석, 인기 키워드 파악, 서비스 품질 향상

3. 개인정보의 보유 및 이용 기간

  • 회원 탈퇴 시: 지체 없이 파기합니다.
  • 뉴스 기사 데이터: 수집일로부터 30일 후 자동 삭제됩니다.
  • 활동 기록: 최근 90일간 보관 후 자동 삭제됩니다.
  • 커뮤니티 게시글: 회원 탈퇴 또는 직접 삭제 시 파기됩니다.
  • FCM 토큰: 로그아웃 또는 회원 탈퇴 시 즉시 삭제됩니다.

단, 관련 법령에 따라 보관이 필요한 경우 해당 기간 동안 보관합니다.

4. 개인정보의 제3자 제공

Kiping은 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 서비스 운영을 위해 다음의 외부 서비스를 이용합니다:

서비스 제공 업체 이용 목적 제공 항목
인증 및 데이터베이스 Supabase Inc. 회원 인증, 데이터 저장 이메일, 프로필, 키워드, 활동 데이터
푸시 알림 Google (Firebase Cloud Messaging) 푸시 알림 전송 FCM 토큰, 알림 내용
AI 분석 Anthropic 뉴스 관련도 분석, AI 리포트 생성 뉴스 기사 제목/내용 (비식별 데이터)
소셜 로그인 Google, Kakao 간편 로그인 인증 토큰 (OAuth)
미디어 저장 Supabase Storage 커뮤니티 이미지/영상 저장 업로드 미디어 파일

5. 개인정보의 파기 절차 및 방법

  • 이용 목적이 달성되거나 보유 기간이 경과한 개인정보는 지체 없이 파기합니다.
  • 전자적 파일: 복구 불가능한 방법으로 영구 삭제합니다.
  • 커뮤니티 미디어 파일: 게시글 삭제 시 스토리지에서 함께 삭제됩니다.

6. 이용자의 권리 및 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:

  • 개인정보 열람 요청: 앱 내 설정에서 본인의 정보를 확인할 수 있습니다.
  • 개인정보 수정: 프로필, 키워드, 알림 설정 등을 직접 수정할 수 있습니다.
  • 회원 탈퇴: 앱 내 설정에서 계정 삭제를 요청할 수 있으며, 모든 데이터가 즉시 삭제됩니다.
  • 알림 수신 거부: 앱 내 설정 또는 기기 설정에서 푸시 알림을 비활성화할 수 있습니다.
  • 게시글 삭제: 본인이 작성한 커뮤니티 게시글 및 댓글을 직접 삭제할 수 있습니다.

7. 개인정보의 안전성 확보 조치

Kiping은 이용자의 개인정보를 안전하게 보호하기 위해 다음과 같은 조치를 취합니다:

  • 모든 데이터 전송 시 SSL/TLS 암호화 적용
  • 데이터베이스 접근 시 행 수준 보안(Row Level Security) 적용
  • API 호출 시 JWT 기반 인증 적용
  • 비밀번호는 저장하지 않음 (소셜 로그인만 지원)
  • 서비스 키 및 민감 정보는 환경 변수로 안전하게 관리

8. 쿠키 및 자동 수집 장치

Kiping 모바일 앱은 쿠키를 사용하지 않습니다. 서비스 이용 과정에서 앱 사용 통계(화면 방문 횟수)가 자동으로 수집되며, 이는 서비스 개선 목적으로만 사용됩니다.

9. 만 14세 미만 아동의 개인정보

Kiping은 만 14세 미만 아동의 회원가입을 제한하지 않으나, 아동의 개인정보를 별도로 수집하지 않습니다. 법정대리인이 아동의 개인정보 처리에 동의하지 않는 경우, 해당 아동의 계정 삭제를 요청할 수 있습니다.

10. 개인정보처리방침의 변경

본 개인정보처리방침은 관련 법령 및 서비스 변경에 따라 수정될 수 있습니다. 변경 시 앱 내 공지 또는 본 페이지를 통해 안내합니다.
